DATA PREPROCESSING & TRAINING

Preprocessing

Dataset cleaned using mean imputation for NaN values. Features normalized using Standard Scaler to standardize distributions (Mean=0, Variance=1) prior to modeling.

Train/Test Split

Rigorous 80/20 data split. 80% (1,279 samples) used for training the ensemble models. 20% (320 samples) strictly reserved for blind validation to prevent overfitting.
